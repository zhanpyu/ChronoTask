# ChronoTask

A productivity web app combining task management, time tracking, and goal planning in a single dashboard.

## What it does

ChronoTask brings together the tools you need to stay organized and focused: manage tasks across a Kanban board, track time with a smart timer, plan your day with a calendar view, and set long-term goals — all in one place.

## Features

- **Task management** — Create, assign, and track tasks with status updates
- **Kanban board** — Visual workflow management with drag-and-drop
- **Smart timer** — Time tracker linked to tasks for accurate time logging
- **Calendar view** — Daily and weekly schedule planning
- **Goals tracking** — Set and monitor long-term objectives
- **Dashboard overview** — Activity feed, assigned tasks, and productivity widgets
- **Authentication** — User accounts with onboarding flow

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
cd ChronoTask/project
npm install
npm run dev
```
